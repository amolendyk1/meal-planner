# Meal Planner Pro - Project Repo

**Client:** Myself

Brief Description
- This repo contains all the files needed to run Meal Planner Pro, a fully client‑side weekly meal‑planning tool. It includes a home planner view, a meal library for adding recipes and food items, a nutrition summary page, and a meal picker for assigning meals to each day of the week. Users can add meals, plan breakfast/lunch/dinner for all seven days, view weekly calories and protein totals, and reset the app at any time.

What you learned
- From this project, I learned how to build a complete multi‑view web application using only vanilla JavaScript, HTML, and CSS. I learned how to manage state, update the DOM dynamically, and persist data using localStorage. I also learned how to create interactive features such as modal windows, navigation between views, and automatic nutrition calculations. This project helped me understand how to structure JavaScript for a real tool, how to debug DOM issues, and how to design a simple, intuitive user experience.
